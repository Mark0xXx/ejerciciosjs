¿Qué puedo hacer?

1. Al clicar un botón, que se ejecute un evento, por ejemplos que se abra una ventana con información para el usuario

2. Cambiar las propiedades de un elemento, por ejemplo cambiar la imagen de fondo de un botón

En este caso Javascript cambiará el atributo src de una imagen

2. Cambiar las propiedades de un elemento, por ejemplo cambiar la CSS de un elemento

3. Obtener información que el PC donde se está ejecutando Javascript tiene. Por ejemplo, la fecha y hora en el momento en que clicas un botón.

4. Operaciones matemáticas. Por ejemplo sumar 1 a un número a cada clic de botón
